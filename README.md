# Raster vs. Vector

Created by Gerard Lucas for IT1600-01 Fall 2022. The project uses vuejs & vite

Project is hosted at https://rv.gerardlucas.com

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
